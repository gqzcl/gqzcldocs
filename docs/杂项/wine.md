[lutris](https://lutris.net/downloads/)

可以运行win游戏在linux上

lutris是一个开放的 Linux 游戏平台。它可以帮助你在一个统一的界面中安装和管理你的游戏。这种支持包括管理 Windows 游戏(通过 Wine 运行)。

[Glintwine](https://bitbucket.org/andycellar/glintwine/src/master/)

一个基于 python 的 GUI 工具，用于管理 Wine 的注册表项。

[play on linux](https://www.playonlinux.com/en/download.html)

这个工具的目的是让用户更容易安装 Windows 软件，比如 Windows 魔兽世界，Adobe Photoshop，Guild Wars 等等。
