# 安装 windows Terminal

## 在 windows Terminal 中设置[Powerline](https://docs.microsoft.com/zh-cn/windows/terminal/tutorials/powerline-setup)

###  安装PowerShell

> github 链接->[here](https://github.com/PowerShell/PowerShell)

### 安装powerline字体

> github 链接->[here](https://github.com/microsoft/cascadia-code)

### 在PowerShell 中设置Powerline

```sh
Install-Module posh-git -Scope CurrentUser
Install-Module oh-my-posh -Scope CurrentUser
```
### 在PowerShell 配置文件中添加

```
Import-Module posh-git
Import-Module oh-my-posh
Set-PoshPrompt -Theme Paradox(主题)
```
> oh-my-posh文档->[here](https://ohmyposh.dev/docs)

### 关于执行策略
参考[here](https://docs.microsoft.com/zh-cn/powershell/module/microsoft.powershell.core/about/about_execution_policies?view=powershell-7.1#managing-the-execution-policy-with-powershell)

更改执行策略
```sh
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
```

### 关于如何打开 PowerShell 配置文件
参考[here](https://docs.microsoft.com/zh-cn/powershell/module/microsoft.powershell.core/about/about_profiles?view=powershell-7#how-to-edit-a-profile)

要在当前 PowerShell 主机应用程序中创建当前用户的配置文件，请使用以下命令：

```sh
if (!(Test-Path -Path $PROFILE)) {
  New-Item -ItemType File -Path $PROFILE -Force
}
```
然后编辑配置文件

```
notepad $PROFILE
```

### 查看oh-my-posh主题

> ```Get-PoshThemes```

scoop install starship
